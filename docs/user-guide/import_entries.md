## Импортирование данных заявки (сделана на сайте orgeo.ru)

Текущая версия по умолчанию поддерживает формат «CSV Winorient с представителями».

Формат полей:

* Группа
* ФИО
* Коллектив
* Представитель
* Разряд
    * 0 = б/р
    * 1 = Iю
    * 2 = IIю
    * 3 = IIIю
    * 4 = I
    * 5 = II
    * 6 = III
    * 7 = КМС
    * 8 = МС
    * 9 = МСМК
    * 10 = ЗМС  // В программе будет указано МСМК
* Номер
* Год рождения
* Номер чипа
* Комментарий
