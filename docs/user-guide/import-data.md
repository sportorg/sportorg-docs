* Группа

  * 2 = IIю
  * 3 = IIIю
  * 4 = I
  * 5 = II
  * 6 = III
  * 7 = КМС
  * 8 = МС
  * 9 = МСМК
  * 10 = ЗМС
* Номер
* Год рождения
* Номер чипа

Для отображения части участников, принадлежащих к одной группе или коллективу, можно использовать фильтрацию.
